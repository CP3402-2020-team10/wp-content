## variable-site/_colors.scss

## Color Picker for Site Header and Footer
* Allows the ability to use a color picker in the customization 
* Add setting and control function in inc/customizer.php to create the color picker in the customization tool in wp-admin.
* Add function to js/customizer.js to apply changes to the header and footer.
